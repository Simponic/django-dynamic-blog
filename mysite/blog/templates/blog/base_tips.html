Bash Cheat Sheet

Command Definition Usage
cd Change to a new directory cd $HOME
ls View files in a directory ls -lh
mkdir Make a new directory mkdir new_folder
rm Remove a file rm garbage.txt
mv Move a file mv homework.txt /dev/null
cp Copy a file cp original.txt plagarised.txt
reset Reset bash terminal reset
grep Use regular expressions to find matches in files grep "[a-z]*" file_with_stuff.txt
cat Concatenate files cat thing1.txt thing2.txt > thing3.txt

Git Cheat Sheet

Command Definition Example
git init Initialize a new git repository git init
git clone Clone a remote git repository git clone git@simponic.xyz:/srv/git/dwm-rice .
git add Add files to be staged in commit git add .
git commit Add staged files to a new commit git commit -m "new commit"
git pull Pull files from remote repo git pull origin master
git push Push commits to a remote git repository git push -u origin master
git status View git status git status
git log View git commit history git log

HTML Cheat Sheet

Tag Definition
<html> The wrapping html tag
<head> A place to store header information like css stylesheets, title, etc.
<title> The tage which holds the title of the page
<meta> Stores meta information of the file, such as character set
<body> Contains the actual content of the page
<div> A block-element which has defined styling
<h1> - <h6> Header tags for bigger information. After h4 is a meme
<table> A tage to store information through other <tX> tags
<link> Link another resource, such as css
<img> Inline image tag

CSS Cheat Sheet

Entry Definition Values/Examples
. The class selector .class-name
# The id selector #id-name
h1,body,etc. The element selector; selects all elements of this tag h1,body,html,a,img,etc.
background-color Change the background color of an element rgb(x,y,z), #abcdef, white, black, etc.
text-align Align the contents within an element center, left, right, justify
border Define the style for a border dashed, solid, double, groove, etc.
display Change how to display the elements withing an element flex, absolute, fixed, etc.
padding Change the space around an element; like package peanuts padding: 10px
margin Change the margin width between an element margin-top, margin-right, margin, etc
width Change the width of an element width:100%, width:auto, etc.
font-family Change the font settings for an element monospace, "Sans Serif", etc.
